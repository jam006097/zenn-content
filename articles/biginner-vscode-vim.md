### 1. VS Code Vim拡張機能のインストール 🔌

まずはVS CodeにVimの機能を追加する拡張機能をインストールしましょう。

1.  VS Codeを開き、左側のアクティビティバーにある「拡張機能」アイコン（四角が3つと1つ離れた四角のマーク）をクリックします。
2.  検索バーに「Vim」と入力し、検索結果の中から「Vim」という名前の拡張機能を探します。
3.  見つけたら、「Install」（インストール）ボタンをクリックします。✅

これでVS CodeがVimのキーバインドを認識できるようになります。🎉

![](/images/vscode/vscode-installvim.png)

### 2. 最低限の設定で快適なVim環境を構築 🛠️

Vim拡張機能をインストールしただけでも基本的な操作は可能ですが、より快適に使うためにいくつかの設定を行うことをお勧めします。

VS Codeの設定ファイル`settings.json`を開きましょう。⚙️
vscode上部の検索窓で`settings.json`を調べれば出てきます。

以下の設定項目を`settings.json`に追記または変更してください。

```json
{
  // --- Vim 拡張の最低限設定 ---
  "vim.useSystemClipboard": true,

  // ESC を押しやすくする
  "vim.insertModeKeyBindings": [
    {
      "before": ["j", "j"],
      "after": ["<Esc>"]
    }
  ],
```

**設定のポイント:** 💡

*   **`"vim.useSystemClipboard": true`**
Vim の yy（コピー）や p（貼り付け）を、OS のクリップボードと共有する設定です。
これにより、Vim でコピーした内容をそのままブラウザやターミナルに貼り付けられます。📋
初心者が最初につまずきやすいポイントなので、必須の設定です。

*   **`"vim.insertModeKeyBindings"`**
挿入モードからノーマルモードに戻る際、通常は Esc キーを使いますが、
キーボードの端にあるため押しにくいと感じる人もいます。
この設定では jj と連続で入力することで、Esc と同じ動作をするようにしています。🙌
ホームポジションを崩さずに操作できるため、作業が快適になります。

ここでは「最低限」で快適に使うための設定だけを紹介しています。
Vim の細かいカスタマイズは、慣れてきてから少しずつ追加すれば問題ありません。😉

### 3. Vimの基本操作を覚えよう 📚

Vimには「モード」という概念があり、これが初心者が最初に戸惑うポイントかもしれません。しかし、基本はたったの3つのモードです。

1.  **ノーマルモード (Normal Mode)**:
    *   デフォルトのモードです。ファイルを開いたときや`Esc`キーを押すとこのモードになります。
    *   文字入力はできませんが、カーソル移動、文字の削除、コピー、貼り付け、検索などの「コマンド」を実行するモードです。
    *   効率的なテキスト編集の要となるモードです。👑

2.  **挿入モード (Insert Mode)**:
    *   文字を入力するためのモードです。✍️
    *   ノーマルモードから`i` (カーソルの位置から挿入)、`a` (カーソルの次の位置から挿入)、`o` (現在の行の下に新しい行を作成して挿入) などのキーで切り替えます。
    *   このモードでは、普段のエディタと同じように文字入力ができます。
    *   `Esc`キー（または上記で設定した`jj`など）でノーマルモードに戻ります。

3.  **ビジュアルモード (Visual Mode)**:
    *   文字を選択するためのモードです。🎨
    *   ノーマルモードから`v` (文字単位)、`V` (行単位)、`Ctrl+v` (矩形選択) などで切り替えます。
    *   選択範囲を削除、コピー、置換などの操作ができます。
    *   `Esc`キーでノーマルモードに戻ります。

**よく使う基本コマンド (ノーマルモードで実行):** 🧠

*   **カーソル移動:** 🧭
    *   `h`: 左へ移動
    *   `j`: 下へ移動
    *   `k`: 上へ移動
    *   `l`: 右へ移動
    *   `w`: 次の単語の先頭へ
    *   `b`: 前の単語の先頭へ
    *   `0`: 行頭へ
    *   `$`: 行末へ
    *   `gg`: ファイルの先頭へ
    *   `G`: ファイルの末尾へ
    *   `:{行番号} `: 指定した行へ移動（例: `:10`で10行目へ）

*   **挿入モードへの切り替え:** ⌨️
    *   `i`: カーソルの位置から挿入
    *   `a`: カーソルの次の位置から挿入
    *   `o`: 現在の行の下に新しい行を作成して挿入
    *   `O`: 現在の行の上に新しい行を作成して挿入

*   **削除・変更:** 🗑️
    *   `x`: カーソル下の文字を削除
    *   `dw`: カーソルから単語の終わりまで削除
    *   `dd`: 現在の行を削除
    *   `d{移動コマンド}`: 移動コマンドで指定した範囲を削除（例: `d$`でカーソルから行末まで削除）

*   **コピー・貼り付け:** 📋
    *   `yw`: カーソルから単語の終わりまでコピー
    *   `yy`: 現在の行をコピー
    *   `y{移動コマンド}`: 移動コマンドで指定した範囲をコピー
    *   `p`: カーソルの後ろに貼り付け
    *   `P`: カーソルの前に貼り付け

*   **その他:** ✨
    *   `u`: 元に戻す (Undo)
    *   `Ctrl+r`: やり直し (Redo)
    *   `:w`: 保存
    *   `:q`: 閉じる
    *   `:wq`: 保存して閉じる

*   **検索・置き換え:** 🔎
    *   `/検索したい文字列`: 下方向へ検索
    *   `n`: 次の検索結果へ移動
    *   `N`: 前の検索結果へ移動
    *   `:%s/置換前/置換後/g`: ファイル全体で「置換前」を「置換後」にすべて置き換え
